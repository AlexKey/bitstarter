2nd Screen JS allows front end developers to easily create applications that run on more than one screen. For example an online racing game on a PC that uses a persons mobile phone as a controller.

Examples could include:

  1. Game controller
  2. TV Remote
  3. Heads up display

FAQ

What is a 2nd screen?
A 2nd screen is a term to describe using more than one device to view or interact with an application. For example a person may be watching youtube on their pc and using their phone to controll the play / pause and playlist.
Can I try it for free?
There is a free usage tier, which allows you to experiment without using a credit card and then varying tiers depending on usage.
